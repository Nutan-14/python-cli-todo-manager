# Python CLI To-Do List Manager

## 📌 Project Description
This is a Command Line Interface (CLI) To-Do List Manager developed using Python as part of my Junior Python Developer Internship. The application allows users to manage daily tasks efficiently using a simple terminal-based interface.

---

## 🚀 Features
- Add new tasks
- View all tasks
- Remove tasks by ID
- Mark tasks as completed
- Input validation and error handling
- Interactive menu-driven interface

---

## 🧠 Technologies Used
- Python 3
- Basic OOP and functions
- CLI (Command Line Interface)
